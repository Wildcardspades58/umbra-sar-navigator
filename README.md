Umbra SAR Navigator — Simplified Feature List

Open Source STAC Viewer
A lightweight, browser-based viewer for exploring SpatioTemporal Asset Catalogs, initially focused on the Umbra Open Data Catalog.

Umbra Open Data Catalog Integration
Connects directly to Umbra’s public STAC catalog and AWS-hosted SAR data, making it easier to browse real open SAR collections.

Interactive Map Exploration
Displays SAR collection footprints on an interactive Leaflet map with selectable basemaps, footprint filtering, and map navigation tools.

Cloud Optimized GeoTIFF Streaming
Streams Umbra GEC GeoTIFF files directly into the map for visual inspection without requiring desktop GIS software.

SAR Deliverables Browser
Shows available collect assets such as GEC GeoTIFFs, SICD, SIDD, CPHD, quicklooks, metadata files, and direct S3 links.

Quicklook and Raster Overlay Tools
Supports preview overlays and streamed image layers with opacity controls, layer toggles, and map fit tools.

Geographic Analytics
Summarizes loaded collections by country, state or region, platform, polarization, collection date, incidence angle, and product type.

Client-Side SAR Visualization
Runs directly in the browser with no backend server required, making it easy to deploy on GitHub Pages or other static hosting platforms.

COG Display Controls
Includes quality, scale, orientation, and canvas-fit controls to improve in-browser visualization of SAR GeoTIFFs.

GeoTIFF Diagnostics
Provides technical diagnostics for loaded rasters, including image size, bands, CRS, pixel size, bounds, and georeferencing warnings.

STAC Catalog Navigation
Allows users to traverse STAC catalog folders, load individual items, scan collections, and export discovered footprints.

Export Tools
Exports loaded SAR footprints as GeoJSON, metadata as CSV, and geographic analytics as CSV.

Designed for Open Geospatial Exploration
Built as a practical open source tool for analysts, developers, educators, and geospatial enthusiasts working with open SAR data.
